# Security docs

## Contents

- [baseline-checklist.md](baseline-checklist.md) -- OWASP ASVS L1 security baseline checklist

## How to use

The checklist is pre-populated with items. As the project is built, set each item's checkbox and add a one-line note on how it is satisfied (or why it is N/A). Treat unchecked items in shippable areas as debt, not decoration.
